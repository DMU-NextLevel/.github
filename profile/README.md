<p align="center">

**NextLevel**은 번뜩이는 아이디어를 누구나 제약 없이 실현할 수 있도록 돕는  **크라우드 펀딩 플랫폼**입니다.  
관심사 기반 추천, 실시간 진행 상황, 알림 시스템 등 다양한 기능을 통해  
사용자와 창작자 모두를 위한 서비스를 제공합니다.
