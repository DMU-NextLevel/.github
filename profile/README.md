## 프로젝트 소개

**WithU**는 번뜩이는 아이디어를 가진 누구나 제약 없이 꿈을 실현할 수 있도록 돕는 **혁신적인 크라우드펀딩 플랫폼**입니다.

### 핵심 가치
-  **개인화된 추천**: 관심사 기반 맞춤형 프로젝트 추천
-  **낮은 진입장벽**: 누구나 쉽게 시작할 수 있는 펀딩 환경
-  **투명한 소통**: 창작자와 후원자 간의 원활한 커뮤니케이션

###  해결하고자 한 문제점

```
기존 펀딩 플랫폼의 한계
├── 높은 진입장벽과 복잡한 프로세스
├── 부족한 개인화 추천 시스템
├── 제한적인 홍보 효과
└── 비효율적인 소통 구조
```

###  우리의 솔루션

- **스마트 추천 알고리즘**: 사용자 관심사 분석을 통한 맞춤형 프로젝트 제안
- **실시간 알림 시스템**: 중요한 업데이트를 놓치지 않는 즉각적 소통
- **간편한 프로젝트 등록**: 직관적인 UI/UX로 누구나 쉽게 시작
- **투명한 진행 현황**: 실시간 펀딩 상태 및 통계 제공

---

### 상세 기능

- ** 직관적인 프로젝트 생성 도구**: 복잡한 과정 없이 간단하게 펀딩 프로젝트 생성
- ** 안전한 결제 시스템**: Toss Payment API를 활용한 신뢰할 수 있는 결제 환경
- ** 강화된 보안**: JWT 인증과 패스워드 암호화로 사용자 정보 보호
- ** 소셜 로그인**: OAuth2 기반 간편 로그인 지원

<div align="center">

| 기능                   | 설명                      |

| **Email 발송**         | SMTP 방식의 Java 이메일 발송 기능 |
| **JWT**              | JWT를 활용한 사용자 인증 및 보안 처리 |

### 핵심 기술 구현

| 보안 & 인증 | 결제 시스템 | 커뮤니케이션 |
|:---:|:---:|:---:|
| **OAuth2 소셜로그인**<br>간편하고 안전한 인증 | **Toss Payment API**<br>신뢰할 수 있는 결제 시스템 | **SMTP 이메일 발송**<br>자동화된 알림 서비스 |
| **JWT 토큰 인증**<br>세션 관리 최적화 | **실시간 결제 상태**<br>투명한 거래 추적 | **실시간 알림**<br>즉각적인 업데이트 전달 |
| **비밀번호 암호화**<br>개인정보 보호 강화 | **환불 시스템**<br>안전한 거래 보장 | **다채널 알림**<br>이메일, SMS 등 |

## 보안 정책

우리는 사용자의 데이터 보안을 최우선으로 생각합니다.

```
🔐 보안 체계
├── 🌐 전 구간 HTTPS 암호화 통신
├── 🔑 JWT 기반 사용자 인증 시스템  
├── 🏛️ 민감 정보 암호화 저장
├── 📊 접근 권한 세분화 및 로그 관리
└── 🔍 정기적 보안 검사 및 모니터링
```
